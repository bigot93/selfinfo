# 첫 번째 블로그 포스트

안녕하세요! 이것은 첫 번째 블로그 포스트입니다.

## 소개

이 블로그는 Next.js로 만들었으며, 다음과 같은 기능을 제공합니다:

- 마크다운으로 글 작성
- 태그 및 카테고리 분류
- 월별 아카이브
- 아름다운 UI

## 마크다운 지원

이 블로그는 마크다운을 완전히 지원합니다:

- **굵은 글씨**
- *기울임*
- `코드 블록`
- [링크](https://nextjs.org)

### 코드 예제

```javascript
function hello() {
  console.log('Hello, World!');
}
```
